# 🌤 Weather App Pro

A modern desktop weather application built using Python and CustomTkinter.

## Features

- Search weather by city
- Real-time OpenWeather API
- Temperature
- Feels Like
- Minimum Temperature
- Maximum Temperature
- Humidity
- Pressure
- Wind Speed
- Visibility
- Sunrise & Sunset
- Weather Description
- Dynamic Weather Icons
- 5-Day Forecast
- Search History
- Favorite Cities
- Dark / Light Mode
- Refresh Weather
- Professional Dashboard
- GitHub Ready

---

## Installation

Clone repository

```bash
git clone https://github.com/yourusername/weather-app-pro.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Open

```python
config.py
```

Replace

```python
API_KEY="YOUR_OPENWEATHER_API_KEY"
```

Run

```bash
python app.py
```
